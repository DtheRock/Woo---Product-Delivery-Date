Plugin to customize product delivery date

== Description ==

Product Delivery Date is a plugin that allows you to customize the delivery date of a product. It adds a delivery date field to the checkout page and saves the delivery date to the order meta data. It also adds a settings page where you can configure the minimum and maximum delivery time.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/customize-product-delivery-date` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Delivery Date screen to configure the plugin
